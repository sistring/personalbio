## Working with AI

CB-Essay ships three files that help AI coding assistants (Claude Code, Cowork, Cursor, Copilot, Codex, and others) work correctly within the framework's data-driven architecture:

- **`AGENTS.md`** — the authoritative ruleset agents read automatically: the customization hierarchy, what never to edit, the essay includes, and the theming system.
- **`CLAUDE.md`** — a one-line pointer that imports `AGENTS.md` so Claude tools get the same rules.
- **`HUMANS.md`** — a guide for *you* on prompting, reviewing AI changes, and going bigger safely.

Start with `HUMANS.md` to learn how to direct an agent; keep architecture rules in `AGENTS.md` so the files don't drift.
